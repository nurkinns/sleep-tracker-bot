🤖 Telegram-бот для трекинга сна и бодрствования (aiogram 3.x + SQLite).
- Кнопки «Уснуть» и «Проснуться» замеряют время сна, выводят статистику и меняют язык.
- Настроен прокси-обход сетевых таймаутов для стабильной работы на хостинге Amvera.

🚀 Как запустить:
1. Зайдите в Telegram.
2. Введите в поиске юзернейм вашего бота (например, @trackerofsleep_bot).
3. Нажмите кнопку «Старт» (/start) для запуска меню.
