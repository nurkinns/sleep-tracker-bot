🤖 Telegram-бот написанный на python для трекинга сна и бодрствования (aiogram 3.x + SQLite).
- Кнопки «Уснуть» и «Проснуться» замеряют время сна, выводит статистику и меняют язык.
- Базируется на стабильном хостинге Amvera.

🚀 Как запустить:
1. Зайдите в Telegram.
2. Введите в поиске юзернейм нашего бота @trackerofsleep_bot.
3. Нажмите кнопку «Старт» (/start) для запуска меню.
